Go by Example: Recursion

Go soporta funciones recursivas. Aquí hay un ejemplo clásico. 

package main
 
import "fmt"

La función fact se llama a sí misma hasta que alcanza el caso base de fact(0).

 
func fact(n int) int {
    if n == 0 {
        return 1
    }
    return n * fact(n-1)
}
 
func main() {
    fmt.Println(fact(7))
