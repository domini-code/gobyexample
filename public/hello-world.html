Go by Example: Hello World

Nuestro primer programa imprimirá el clásico mensaje “hello world”. Aquí está el código fuente completo.

 
            
              
                
              package main
 
import "fmt"
 
func main() {
    fmt.Println("hello world")
}

Para ejecutar el programa, coloque el código en hello-world.go y utilice go run.

 
$ go run hello-world.go
hello world

A veces querremos integrar nuestros programas en binarios. Podemos hacer esto usando go build.

 
$ go build hello-world.go
$ ls
hello-world    hello-world.go

Luego podemos ejecutar el binario compilado directamente.

 
$ ./hello-world
hello world

Ahora que podemos ejecutar y crear programa Go básico, aprende más sobre Go.
